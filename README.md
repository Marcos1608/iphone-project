# 📱 iPhone (Java Simulation)

Este projeto é uma simulação em Java de algumas funcionalidades do primeiro iPhone, executada diretamente no terminal.

A aplicação tem como objetivo praticar conceitos de Programação Orientada a Objetos (POO), modelando componentes clássicos de um smartphone em forma de classes e interfaces.

# 🎯 Objetivo

O projeto busca representar, de forma simples e didática, três funcionalidades icônicas apresentadas no primeiro iPhone:

🎵 Reprodutor Musical

📞 Aparelho Telefônico

🌐 Navegador de Internet

Cada funcionalidade foi modelada utilizando princípios de abstração, encapsulamento e interfaces, simulando como diferentes módulos podem compor um único dispositivo.

# 🧠 Conceitos de POO Utilizados

Interfaces

Implementação de métodos

Separação de responsabilidades

Modelagem de objetos

Organização modular de classes

O objetivo principal não é replicar um smartphone real, mas treinar a estruturação de código orientado a objetos em Java.

# ⚙️ Funcionalidades
## 🎵 Reprodutor Musical

Permite simular funções básicas de um player de música:

Tocar música

Pausar música

Selecionar música

## 📞 Aparelho Telefônico

Simula funções básicas de um telefone:

Fazer ligação

Atender ligação

Iniciar correio de voz

## 🌐 Navegador de Internet

Simula um navegador simples:

Exibir página

Adicionar nova aba

Atualizar página

#▶️ Como Executar

1. Clone o repositório
```
git clone https://github.com/seu-usuario/iphone.git
```
2. Entre na pasta do projeto
```
cd iphone
```
3. Compile o projeto
```
javac *.java
```
4. Execute
```
java Iphone
```
# 💻 Tecnologias Utilizadas

- Java

# 📚 Motivação

Este projeto foi desenvolvido como exercício prático de POO em Java, inspirado no conceito do primeiro iPhone, que reunia múltiplas funcionalidades em um único dispositivo.

A ideia é demonstrar como interfaces e classes podem representar diferentes capacidades de um mesmo objeto.
